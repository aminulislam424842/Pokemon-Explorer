# ⚡ Pokémon Explorer

A clean and modern Pokémon Explorer UI built with **HTML5** and **CSS3**. This project showcases a simple Pokémon interface with a navigation bar, search section, Pokémon cards, feature cards, and a footer.

---

## ✨ Features

- 🎮 Modern Pokémon-inspired UI
- 🔍 Search bar with search icon
- 🎯 Filter button
- 🃏 Pokémon cards
- ❤️ Favorite button
- 🏷️ Pokémon type badges
- 📦 Feature section
- 📱 Clean responsive-ready layout
- 🦶 Footer with Pokémon watermark

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
